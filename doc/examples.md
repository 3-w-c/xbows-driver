# Examples

```yaml
---

layer: custom 1

keymap:
  CapsLock: Escape
  Windows:
    layer: 3
  Space_L: Windows
  Enter_M: Equal
  Shift_M: BackSpace_R

lights:
  animation:
    - all
  pattern:
    - color: [77,77,ff]
      type: monochrome
```

```yaml
---

layer: custom 2

keymap:
  H: Left
  J: Down
  K: Up
  L: Right
  M: MediaPause
  Comma: MediaLast
  Period: MediaNext
  Left: MediaVolUp
  Right: MediaVolDown

lights:
  animation:
    - all
  pattern:
    - color: [55,55,ff]
      type: monochrome
```
